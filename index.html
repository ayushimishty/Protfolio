I'm a UI/UX Designer and Frontend Developer with 6 years of experience crafting user-centred digital products across web and mobile platforms. I work across the full design lifecycle — from requirement gathering, user research, and information architecture through to high-fidelity Figma prototypes and frontend implementation.

My strength lies in bridging the gap between design and development. I speak both languages fluently — I can hand off a prototype and then build the component myself. This means fewer translation errors, faster iteration, and products that actually match what was designed.

I've worked across enterprise CRM systems, fintech dashboards, real estate platforms, field-service operations tools, and consumer e-commerce — building reusable design systems and responsive interfaces that scale.

Design Process & Skills

How I work,
and what I know

01

Discover & Research

Stakeholder interviews, requirement gathering, competitive analysis, and user journey mapping. Understanding the real problem before touching a pixel.

02

Define & Architect

Information architecture, user flows, and low-fidelity wireframes. Establishing the skeleton before the skin — structure first, aesthetics second.

03

Design & Prototype

High-fidelity UI in Figma or Adobe XD. Interactive prototypes that stakeholders can validate before a single line of code is written.

04

Build & Refine

Pixel-perfect frontend implementation in HTML and CSS — bridging the gap between design files and production. Iterate based on feedback until it's right.
